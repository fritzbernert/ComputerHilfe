Download Seite der Anwendung öffnen

Auf der Webseite der gewünschten Anwendugn befindet sich meistens eine Button oder ein Link der zum Download führt.
Dabei sollte man darauf achten das man auf keine Werbung klick die oft wie ein Download Button aussieht.


Installations Datei herunterladen

Bei vielen Programmen wird automatisch die Installationsdatei herunter geladen.
Bei manchen Programmen muss man allerdings vorher eine Version auswählen.
Bei Windows Systemen wählt man also eine Windows Version aus.

Oft gibt es dabei noch eine x32 und eine x64 version.
Falls sie nicht wissen welche Windows version sie haben können die das so herausfinden. Welche Windows Version benutze ich? (x32(bit) oder x64(bit))
Wählen sie die richtige Windows version aus und folgen sie weiteren möglichen Anweisungen auf der Webseite um den Download zu starten.


Wo finde ich den fertigen Download?

Um den fertigen Download zu finden Windowstaste+E gleichzeitig drücken.
Explorer Fenster öffnet sich.
In der Sidebar Links oben auf Downloads klicken.
Download sollte sich ganz oben als .exe/.msi Datei befinden.

Download ist nicht im Downloads Ordner?

Im Browser STRG+J gleichzeitig drücken.
Neues Fenster mit allen Downloads öffnet sich.
Neben Download befinded sich ein Ordner Symbol oder auf anderen Browsern ein Button der zum Downloadordner führt.
Linksklick auf das Symbol oder Link.
Ordner in dem sich der Download befindet öffnet sich.


Anwendung installieren

Wenn die Installation nicht automatisch nach dem Download gestartet wurde linksklicken sie zweimal schnell auf die heruntergeladene Installationsdatei.
Ein Installationsfenster sollte sich öffnen.
Folgen sie allen Schritten der Installation.
Wenn die Installation beginnt wird sich ein Fenster öffnen um die Installation zu bestätigen.
Linksklicken sie in diesem Fenster auf Ja um die Installation zu starten.


Anwendung ausführen

Wenn die Installation abgeschlossen ist können sie die neue Anwendung finden indem sie einen Linksklick auf das Windowssymbol unten links machen.
Die neue Anwendung sollte sich oben links unter kürzlich hinzugefügt befinden.
Alternativ können sie auch nach dem klicken auf das Windowssymbol einfach den Namen des neuen Programms eingeben um danach zu suchen
und wenn es gefunden wurde einen Linksklick auf die anwendung machen um sie zu starten.
